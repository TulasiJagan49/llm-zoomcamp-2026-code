# llm-zoomcamp-2026-code

This repo has code for the llm-zoomcamp course work of June 2026 cohort.


I am currently using a local model downloaded via ollama with openai library. 

We can download the models from here: https://ollama.com/library after installing ollama locally.

Examples to integrate downloaded ollama models with OpenAI client can be found here: https://docs.ollama.com/api/openai-compatibility
